About this service

What we install

Ceiling fans look simple but the wiring matters. We check that the existing box is fan-rated (most older Atlanta homes aren't — we'll upgrade to a fan-rated box as part of the install). If the circuit doesn't have a dedicated neutral, we work with what's there or suggest a smart-fan control that doesn't need one.

Recessed lighting retrofits are a before-and-after moment for most rooms. We cut in, wire, trim, and aim every can. The modern LED trims pull about one-sixth the wattage of old incandescent cans, so you can add more lights without upsizing the circuit.

What to expect on install day

Chandelier and pendant hangs look easy until you're hanging something heavy over a dining room table. We add proper blocking and fan-rated boxes where needed, and we level the fixture to the table — not the ceiling.

Outdoor fixtures (sconces, spotlights, porch ceiling fans) need weatherproof boxes and GFCI protection. We handle the fixture install plus any rewiring needed to bring an outdoor circuit up to current code.

What to expect

  1. Send us a photo of the location and what you want installed — we'll give you a ballpark over text.
  2. Schedule an install window. Most single-fixture jobs take 60–90 minutes.
  3. We do the install, test the switch, clean up, and take the old fixture/box with us.
  4. Done. Usually done the same visit unless we discover a wiring surprise.

FAQ

Common questions about Ceiling Fans & Lighting.

How much does ceiling fan installation cost?

A standard ceiling fan install (existing fan-rated box, working switch) is $125–$175. If the box needs upgrading to fan-rated, add $75. If we need to run a new switch leg, that's quoted separately.

Do you install lights I buy from Home Depot or Amazon?

Yes. We'll install whatever fixture you buy. If the instructions are missing or the fixture is defective, we'll pause and let you know — won't force-install something that's broken out of the box.

Can you do recessed lighting retrofits?

Yes — new-construction cans or retrofit trims in existing ceilings. We cut in, wire, and aim every light. Typical 6-can living room install is 3–4 hours.
